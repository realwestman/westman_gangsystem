Gang System
-----------
Introducing a new and highly optimized gang system script includes all essential features for immersive roleplay: wardrobe, garage, armory, boss menu, and more. Built with performance in mind, it’s easy to configure and integrates seamlessly into any RP

Features
--------
Gang House Blip: Each gang base is clearly marked on the map with customizable blip icons to enhance visibility and organization for the gang members

Wardrobe System: Members can easily change outfits or manage saved uniforms from a clean, in-game UI, tailored to gang roleplay.

Garage Integration: Shared gang vehicles with a built-in garage system, allowing smooth vehicle retrieval and storage.

Armory Access: Gangs can purchase weapons and items through a secure and intuitive armory menu.

Shared Stash: A centralized inventory stash allows members to store and retrieve gang-related items quickly and efficiently.

Boss Menu: You've the option to add your bossmenu export to the function.

Requirements
----
Your server needs to be based on ESX

Oxmysql needs to be installed and integrated

You’ll need to have `esx_menu_default` installed.
